## Tehnologii utilizate

- **Frontend**: React + Vite + Tailwind CSS
- **Backend CMS**: Strapi (Headless CMS)
- **Hosting Frontend**: Netlify
- **Hosting Backend**: Strapi Cloud
- **Versioning**: GitHub

---

## Structura site-ului

- **Home** - Hero Banner, Hero Text, carduri articole
- **Despre Noi** - Pagina About (Single Type din Strapi)
- **Articole** - Lista articole (Collection Type din Strapi)
- **Contact** - Formular contact, adresă, telefon

---

## Funcționalități

- Meniu desktop sticky
- Meniu mobil hamburger + offcanvas
- Light/Dark mode toggle
- Design responsive
- 3 fonturi Google: Playfair Display, Lato, Merriweather
- Temă Tailwind: Solar Dusk (tweakcn.com)
